# React Frontend (Smart Household Manager)

This folder contains the React + Vite frontend for the Smart Household Manager project.

## Purpose
- Hosts the React-based Home Dashboard
- Implements component-based UI architecture
- Gradually replaces the legacy HTML/JS frontend

## Tech Stack
- React
- Vite
- JavaScript
- CSS

## Status
Actively in development. UI-first approach with incremental backend integration.
